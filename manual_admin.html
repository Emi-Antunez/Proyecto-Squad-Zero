Introducción al Panel de Administración

¿Qué es un Administrador?

Un administrador es un usuario con permisos especiales para gestionar completamente el sistema de Tour Náutico Colonia.

Permisos de Administrador:
  • Ver todas las reservas del sistema
  • Gestionar usuarios (crear, modificar, eliminar)
  • Cambiar roles de usuarios
  • Crear y gestionar publicaciones
Funciones Adicionales:
  • Cancelar reservas de cualquier usuario
  • Exportar datos a Excel
  • Generar reportes
  • Acceso completo al sistema

Acceso al Panel de Administración

¿Cómo Acceder?

Paso 1: Inicia sesión con tu usuario administrador
Paso 2: Serás redirigido automáticamente al Panel de Administración
URL: http://localhost/Proyecto-Squad-Zero/admin.html
Seguridad: Solo usuarios con rol "admin" pueden acceder. Si un usuario normal intenta acceder, será redirigido con el mensaje: "Acceso denegado. Solo administradores pueden acceder a esta página."

Filtrar y Buscar Reservas

Selecciona un tour específico para ver solo esas reservas:

  • Navegación Bahía
  • Riachuelo
  • Anchorena
  • Todos los tours

Busca reservas por nombre, apellido, email o usuario del cliente.

La búsqueda es en tiempo real y no distingue mayúsculas/minúsculas.

Ordena las reservas por fecha (más reciente o más antigua).

Primera vez: Más antigua → Más reciente

Segunda vez: Más reciente → Más antigua

Cancelar Reserva de Usuario

Importante: Siempre escribe un motivo claro y profesional cuando canceles una reserva.
Pasos para cancelar:
  1. Busca la reserva en la lista
  2. Click en "Cancelar" (botón rojo)
  3. Escribe el motivo de cancelación
  4. Confirma la cancelación
Resultado: El usuario recibe un email automático con el motivo de cancelación.
Ejemplos de motivos válidos:
  • "Condiciones climáticas adversas previstas"
  • "Mantenimiento programado del barco"
  • "Cancelación solicitada por el cliente"

Eliminar Usuario

⚠️ ADVERTENCIA: Esta acción es IRREVERSIBLE

Al eliminar un usuario:

  • Se elimina el usuario permanentemente
  • Se eliminan todas sus reservas
  • No se puede recuperar la información
Cuándo eliminar: Usuario solicita eliminación, cuenta duplicada, usuario fraudulento, limpieza de base de datos.

Gestión de Publicaciones

Crear Nueva Publicación

Publica noticias, promociones y anuncios en la página principal.

Campos requeridos:
  • Título: Claro y llamativo
  • Descripción: Contenido completo
  • Fecha: Fecha de publicación
Campo opcional:
  • Imagen: JPG, PNG, GIF, WEBP (máx. 2MB)
Resultado: La publicación aparece inmediatamente en la página principal.

Editar Publicación

Modifica publicaciones existentes.

  1. Busca la publicación en la lista
  2. Click en "Editar"
  3. Modifica los campos deseados
  4. Guarda los cambios

Eliminar Publicación

Nota: La eliminación es permanente y no se puede deshacer.

Usos: Promoción expirada, información desactualizada, error en publicación.

Buenas Prácticas

Títulos efectivos:
  • ✅ Cortos y directos
  • ✅ Incluyen palabras clave
  • ✅ Usan números cuando relevante
Ejemplos:
  • ✅ "¡Última semana! Descuento 30%"
  • ✅ "Nuevo tour: Isla San Gabriel"
  • ❌ "Información"
